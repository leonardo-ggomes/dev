Sobre o Node.js

 Node.js® é um ambiente de tempo de execução JavaScript multiplataforma e de código aberto.

  • Assíncrono
  • Orientado a Eventos
  • Ecalonável
  • Single Thread
  • I/O não bloqueante

Instalação do Node.js

  Acesse o site oficial para ter acesso a última versão do instalador.
  Use a versão LTS. Caso você tenha o Node.js na sua máquina, execute o comando abaixo no terminal para verificar a versão instalada.

C:\Users> node -v

Test-drive

console.log("Hello World")

Documentação Node.js

Criando o primeiro servidor...

const http = require('node:http');
const hostname = '127.0.0.1';
const port = 3000;

const server = http.createServer((req, res) => {
 res.statusCode = 200;
 res.setHeader('Content-Type', 'text/plain');
 res.end('Hello, World!\n');
});

server.listen(port, hostname, () => {
 console.log(`Server running at http://${hostname}:${port}/`);
});
