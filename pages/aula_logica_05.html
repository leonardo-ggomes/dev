Sobre o If/Else

 A condicional if é uma estrutura condicional que executa a afirmação, dentro do bloco, se determinada condição for verdadeira. Se for falsa, executa as afirmações dentro de else.
  • Use o if para executar um bloco de código quando a afirmação for true
  • Use o else para executar um bloco de código quando a afirmação for false ainda na mesma estrutura
  • Use o else if para executar uma nova verificação caso a primeira afirmação seja false

Estrutura do If/Else

  Exemplo de como declarar um bloco if/else.

if(condicao){
  coloque seu código aqui...
}

Exemplo do bloco if

var gostoFilme = true;

if(gostoFilme === true){
  console.log("Sim, eu gosto!") ←
}

Exemplo do bloco if/else

var eMarvel = false;

if(eMarvel === true){
  console.log("É verdade, muito bom!")
} else{
  console.log("Poxa!") ←
}

Exemplo do bloco else if

var heroi = "Hulk"

if( heroi === "Thor" ){
  console.log("Dominante")
} if( heroi === "Hulk" ){
  console.log("Introvertido!") ←
} else{
  console.log("Fraco")
}

If/Else

 Lembre-se que a condição só será satisfeita caso a afirmação resultar em true. No entanto, você pode usar o bloco else para capturar afirmações false.
