Sobre variáveis

 Variáveis é um espaço em memória, destinado ao armazenamento de dados.
  • Variável em javascript pode ser declarada de 4 formas automática, let, var ou const.
  • Váriavel automática é declarada no primeiro uso.
  • Váriavel const não pode ser alterada depois de inciada.
  • Váriavel let tem visiblidade local.
  • Váriavel var tem visiblidade global.
  • Não declare variáveis com acento, caracter especial, começando com número e espaço entre palavras.
  • Use o padrão cammelCase na hora de declarar variáveis. Veja let nomeCompleto.
  • É permitido _ no começo.
  • O nome da variável deve fazer sentido.

Declaração de Variável

  Exemplo de como declarar uma variável. O sinal = é um operador de atribuição, isso significa que o valor a esquerda está sendo passado para a variável a direita. Não confunda com a igualdade.

let assunto = "variável javascript"
let dolarCanadense = 3.75
let qtdePessoas = 10
let status = true
const PI = 3.1416

Tipo de dados

 O javascript suporta alguns tipos de dados, que inclusive são iguais em outras linguagem de programação.
  • string é um tipo de dados textual, use " ".
  • number é um tipo númerico para inteiro 1,2,3,4...., ou decimal 1.1, 2.4, 2.5.
  • boolean é tipo que tem duas possibilidades true ou false.
  • object é um tipo declarado com { } e contém propriedades.
  • array é um tipo que permite armazenar um coleção de dados separados por , entre colchetes [ ].
  • null é do tipo vazio, que representa uma variável sem valor.
  • undefined é um tipo que representa que a variável é sem tipo definido, ou valor.
