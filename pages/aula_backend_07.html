O que é JSON ?


  JSON significa "JavaScript Object Notation" (Notação de Objeto JavaScript). É um formato leve e fácil de ler para representar e trocar dados entre diferentes partes de um programa ou entre aplicativos. JSON é amplamente usado para transmitir informações em APIs da web, armazenar configurações e trocar dados entre o navegador e um servidor.


Como funciona:


 Pense no JSON como uma maneira de escrever informações em um formato que tanto humanos quanto computadores possam entender. É como escrever uma lista de compras usando uma linguagem que um amigo seu também pode entender.


JSON


  A notação JSON possui chave e valor.

{
   "chave":"valor"
}
