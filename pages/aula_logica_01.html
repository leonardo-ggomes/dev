Sobre o While

 Uma declaração while executa suas instruções, desde que uma condição especificada seja avaliada como verdadeira. Use para:

  • Repetir uma lógica quantas vezes quiser
  • Controlar a saída de uma lógica

Estrutura do While

  Exemplo de como declarar o While no seu código. Você pode ter ainda outros while aninhados.

while(condicao){
  coloque seu código aqui...
}

Exemplo

var contador = 0;

while(contador < 3){
  contador++;
}

While

 Lembre-se que a condição deve ser atualizada até que seja false, caso contrário seu loop poderá ficar infinito. No exemplo anterior acrescemos a cada iteração o valor da variável contador a 1.

While

 O teste da condição ocorre antes que o laço seja executado. Desta forma se a condição for verdadeira o laço executará e testará a condição novamente. Se a condição for falsa o laço termina e passa o controle para as instruções após o laço.
