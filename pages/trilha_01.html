Requisitos da Plataforma

  Construa uma plataforma de Reserva de Passagens.
 Visto que você já implementou esse programa anteriormente, agora desenvolva as mesmas funcionalidades na versão web.

  A plataforma deverá ter as seguintes funcionalidades:

  • Check In (Confirmação de Reserva).
  • Consulta de voos.
  • Mapa de assento.
  • Emissão de Ticket.
  • Fale Conosco

Introdução ao DOM

  No arquivo script.js implemente a lógica que será executada nas páginas do projeto.

 Nós podemos acessar os elementos html atráves do javascript. Uma maneira fácil é atráves da API DOM. Geralmente fazemos isso pelo id, class ou name.

Mais sobre DOM

Exemplo

<input id="identificador" type="text">

<script>
  var ipt = document.getElementById("identificador")
</script>

Como o document.getElementById() retorna um valor, precisamos armazenar em uma variável.

Extra

Window Location

  URL é o endereço web que nos leva a um determinado recurso ou site, exemplo www.meusite.com/index.html.
 Nós podemos realizar essa navegação pelo javascript, ou seja, ir de uma página a outra. Use o comando window.location.

<script>
  window.location = "fale_conosco.html"
</script>

Window Location

  Podemos ainda passar valores para a página requisitada. Veja, fale_conosco.html?valor ? indica que você vai passar valores na url.

<script>
  window.location = "fale_conosco.html?1"
</script>

Extra

Location Search

Anteriormente passamos o valor na url www.meusite.com/fale_conosco.html?1.
Agora vamos pegar esse valor e realizar a nossa lógica. Vale ressaltar que esse bloco de código deve ser colocado no header da estrutura html, na página do recurso de destino.

<script>
  var url = location.search

 //Comentário ?1
 var url_val = url.replace('?','')
</script>
