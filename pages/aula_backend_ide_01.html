Sobre IDE

 Um ambiente de desenvolvimento integrado (IDE) é um software para criar aplicações que combina ferramentas comuns de desenvolvedor em uma única interface de usuário gráfica (GUI). Um IDE geralmente consiste em:
  • Editor de Código Fonte
  • Automação de compilação local
  • Debbuger

Mais sobre IDE

  O IDE combina uma série de recursos que facilita o trabalho do desenvolvedor. Dentro de uma interface central é possível editar, compilar, debugar, empacotar código de forma mais eficiente e produtiva.

IDE x Editor de Código


  Vejamos a diferença entre IDE e Editor de Código, além de alguns softwares mais utilizados por desenvolvedores ao redor do mundo.
