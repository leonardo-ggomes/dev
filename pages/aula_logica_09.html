Sobre Arrays


 Um array em JavaScript é uma variável especial que armazena uma coleção ordenada de valores, que podem ser de qualquer tipo de dados. Esses valores são acessados através de índices numéricos, começando por 0.
  • let filmes = [ "Batman", "Barbie", "Top Gun", "Fast and Furious"]

Declaração de Array


  Exemplo de como declarar um array. Os dados são colocados entre [ ] separado por vírgula.

let carros = ["uno","jetta","saveiro","kwid"]

carros[2]

>> A saída será "saveiro"

Pontos para recordar


  • Opte por uma estrutura homogênea.
  • O índice é contado a partir do 0.
  • Use loop para manusear o array.

Imagine que você precise armazenar 30 nomes...

var nome1 = "José";
var nome2 = "Ribas";
var nome3 = "Marcela";
var nome4 = "Carla";
var nome5 = "Joana";
var nome6 = "Edivaldo";
...

  e se fosse 300 ?

O array é a solução.

const nomes = ["Ribas","Marcela","Joana","Carla",...]

Acesse os valores pelo indíce

const nomes = []

nomes[0] = "Ribas"
nomes[1] = "Marcio"
nomes[2] = "Leandro"


for( let i = 0; i < 3; i++ ){
  console.log(nomes[i])
}

>> Ribas, Marcio, Leandro
