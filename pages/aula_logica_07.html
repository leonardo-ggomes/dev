Sobre a Function

 Function é um bloco de código que executa uma determinada tarefa.
  • As functions recebem parâmetros.
  • Retornam dados.
  • São executadas somente quando invocadas.
  • São reutilizadas.

Bloco Function

  Exemplo de como declarar uma function.

function name(parametro1,parametro2){
 //Seu código aqui
}

Exemplo de function

function multiplicar(p1,p2){
 return p1 * p2
}

Function

 Para se definir uma fução use a palavra function seguido pelo nome e o ().
 Entre o par de { } coloque sua lógica. Caso queira passar parâmetros, coloque dentro dos parênteses (parametro1, parametro2) separdo por vírgula.

Return

 A palavra return é reponsável por devolver valor e parar a execução da função.

function multiplicar(p1,p2){
 return p1 * p2
}

var resultado = multiplicar(10,2)

Invocando uma função

 O operador ( ) é reponsável por executar a funcão. Você pode chamar funções em eventos, códigos JavaScript ou auto invoca-lás.
