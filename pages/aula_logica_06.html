Sobre o Switch

 Switch é uma instrução condicional que pode executar diferentes ações dependendo da condição
  • O valor é comparado com o valor de cada caso
  • Se houver uma correspondência, o bloco de código associado será executado.
  • Se não houver correspondência, o bloco de código padrão será executado.

Estrutura do Switch

  Exemplo de como declarar um bloco switch.

switch(expressao){
  case x:
   Seu código aqui
   break;
  case y:
   Seu código aqui
   break;
  default:
   Seu código aqui
}

Exemplo do bloco switch

var filmeEscolhido = "batman";

switch(filmeEscolhido){
  case "spiderman":
   console.log("Vai teia!")
   break;
  case "batman":
   console.log("Te peguei, Coringa!")
   break;
  default:
   console.log("Nenhum filme escolhido.")
}

Switch

 Lembre-se que a palavra break interrompe a execução do código e sai da instrução switch, então se você omitir a instrução break, o próximo caso será executado mesmo que a avaliação não corresponda ao caso.
O bloco  Default é executado quando nenhuma verificação caso for satisfeita.
