O que é API ?


  API significa "Interface de Programação de Aplicativos". É como um conjunto de regras e instruções que permite que diferentes partes de software (como aplicativos, sistemas e serviços) se comuniquem e interajam entre si de maneira organizada e padronizada.


 Uma API define como os diferentes componentes de software podem solicitar e trocar informações, executar ações e usar funcionalidades uns dos outros.


API


 Pense em uma API como um cardápio em um restaurante. Você, como cliente (ou usuário de software), não precisa saber como o chef prepara cada prato. Em vez disso, você usa o cardápio para fazer um pedido, e o chef (ou outro sistema de software) executa o trabalho nos bastidores e fornece a comida pronta para você.

API Node.js


const http = require(http)

const server = http.createServer((req, res) => {
  // Definindo o cabeçalho da resposta
  res.writeHead(200, { 'Content-Type': 'application/json' });

 // Enviando a resposta como JSON
 res.end(JSON.stringify({ status:"ok"}));
})

const port = 3000
server.listen(port, () => {
  console.log(`Servidor rodando em http://localhost:${port}`)
})
