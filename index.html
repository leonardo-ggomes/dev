Calculadora Desktop

  Calculadora feita com o framework Electron JS. Para construção da aplicação foi utilizado html, css e javascript. A disposição dos componentes visuais possuem layout grid. A calculadora realiza operações matemáticas simples como, substração, adição e multiplicação.

Execute a instrução abaixo no seu terminal para baixar as dependências do projeto.

$ npm install
